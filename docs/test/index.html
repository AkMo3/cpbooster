Test

cpbooster provides the following command for you to easily test any program.

cpb test <filePath> [options]

This command will test your program against all available test cases (default behavior) or just one test case if --testId flag is specified. It supports the following veredicts.

Notice that <fileName> should also be the name of the corresponding source file (cpp, py, ...), since cpbooster uses <fileName> to make associations between source files and test cases. See File Structure
